# Blog-Backend

#### Used technologies

* PostgresSql
* Lombok
* Docker
* Swagger
* Log4j2
* Spring Boot
* Spring Data
* Spring Security
* Mail Sender
* Mapstruct
* Vault storage

#### Build app
~~~
mvn clean install -DskipTests
~~~
#### Run app
~~~
java -jar target/blog-0.0.1-SNAPSHOT.jar
~~~

#### All possible requests can be viewed at the link

~~~
http://localhost:8080/swagger-ui/
~~~
